Установка и работа с шаблоном (с правами администратора!):
В терминале:
$ npm i
$ npm run dev -- Запуск шаблона в режиме разработки
$ npm run build -- Компилит все необходимиє файлы min, шрифти, webp(Конечниє файли)
$ npm run zip -- Создает архи с исходниками
$ npm run deploy -- Загружает файли на удаленный сервер

Ошибка Pyton
Решени:
npm install --global windows-build-tools
